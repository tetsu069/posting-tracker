中野区ポスティング踏破チェック

使い方:
1. このフォルダ内の index.html / areas.geojson / roads.geojson をGitHub PagesなどHTTPSで公開します。
2. スマホでURLを開きます。
3. 「開始」を押して歩くと、軌跡が青、踏破済み道路が緑になります。
4. 同じスマホ・同じブラウザなら、途中で閉じても記録はブラウザ内に残ります。
5. 記録DLでバックアップJSONを保存できます。

注意:
- GPSはHTTPS公開環境でないと動かない場合があります。
- 踏破判定は道路から18m以内です。厳しすぎ/甘すぎる場合は index.html の threshold = 18 を調整してください。
